Sistema de Gestión de Países en Python
Descripción:

Este proyecto fue desarrollado como Trabajo Integrador de la materia Programación. Consiste en un sistema de gestión de información de países implementado en Python que permite almacenar, consultar y modificar datos a partir de un archivo CSV.

El programa permite:
- Mostrar todos los países registrados.
- Agregar nuevos países.
- Actualizar información existente.
- Buscar un país específico.
- Filtrar países por distintos criterios.
- Ordenar los registros.
- Obtener estadísticas básicas.
- Guardar los cambios realizados en el archivo CSV.

Requisitos
Python 3.x
No es necesario instalar librerías externas, ya que el programa utiliza únicamente funciones incorporadas en Python.

Archivos del proyecto
Trabajo Integrador Obligatorio.py: código fuente principal.
paises.csv: archivo que contiene los datos de los países.
Documentación.pdf: informe del proyecto.
README.md: descripción e instrucciones de uso.

Instrucciones de uso:
Descargar o clonar el repositorio.
Asegurarse de que el archivo paises.csv se encuentre en la misma carpeta que el programa.
Ejecutar el archivo:
python "Trabajo Integrador Obligatorio.py"
Seleccionar las opciones del menú para interactuar con el sistema.
Funcionalidades:
- Mostrar países.
- Agregar país.
- Actualizar país.
- Buscar país.
- Filtrar países.
- Ordenar países.
- Mostrar estadísticas.
- Guardar cambios.
- Salir.
